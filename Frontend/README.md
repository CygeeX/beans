# 豆丰智测 - 大豆产量智能分析系统

## 项目简介

这是一个面向普通用户的大豆产量预测系统前端界面。用户无需了解任何技术细节，只需上传数据文件，即可获得智能产量预测结果。

## 功能特性

### 1. 结果上传页面（/upload）
- 支持两种分析模式：
  - **快速预测**：仅需上传地面观测数据
  - **训练模型**：需上传地面观测数据 + 真实产量记录
- 支持拖拽和点击上传文件
- 实时显示上传和分析进度
- 分析完成后自动跳转到结果详情页

### 2. 结果详情页面（/result 或 /result/:runId）
- 展示历史分析记录列表
- 查看特定分析的详细结果
- 下载分析报告文件
- 支持重新分析

## 页面路由

| 路径 | 页面名称 | 功能描述 |
|------|---------|---------|
| `/` | 首页 | 自动重定向到 `/upload` |
| `/upload` | 结果上传页面 | 上传数据文件并触发分析 |
| `/result` | 结果列表页面 | 展示所有历史分析记录 |
| `/result/:runId` | 结果详情页面 | 展示特定分析的详细结果 |

## 安装与运行

### 1. 安装依赖
```bash
cd my-project
npm install
```

### 2. 配置后端 API 地址
编辑 `.env` 文件，设置后端 API 地址：
```
VITE_API_BASE_URL=http://localhost:8000
```

### 3. 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173

### 4. 构建生产版本
```bash
npm run build
```

构建产物将生成在 `dist/` 目录。

## 技术栈

- **框架**：Vue 3 (Composition API)
- **路由**：Vue Router 4
- **构建工具**：Vite
- **样式**：原生 CSS（Scoped）
- **HTTP 请求**：Fetch API
